１　はじめに
　リレーショナル型データベース実習用にhtaのSQL操作画面を作りました。ファイル選択枠で選んだ任意のアクセスデータベースに、テキストエリアにSQL命令を打ち込み、その実行結果をウインドウ内に表示させます。
　どの.mdbファイルでも使えますが、予め３種類の教材用データベースを創作しましたので、それを選べばすぐに実習できます。

２　同封ファイル
　アクセスデータベースＳＱＬ操作画面.hta
　初歩のＳＱＬ操作法.pdf
　取引データベース.mdb
　所有ＣＤデータベース.mdb
　動物貸出予定データベース.mdb

３　使い方
　"アクセスデータベースＳＱＬ操作画面.hta"を選び、操作画面を呼び出します。［操作するアクセスデータベースファイルのパス］のファイル選択ボタンを押して、フォルダを選び、拡張子".mdb"のアクセスデータベースファイルをクリックして決定します。［実行させたいＳＱＬ命令］のテキストエリアにSQL命令を打ち込み、［ＳＱＬ実行］ボタンを押します。するとこの［ＳＱＬ実行］ボタンの下にその実行結果が表示されます。
　もし、間違えた命令文を打ち込んだら、デバッグ用ダイアログが表示されたり、エラーメッセージが表示されたりします。反応しなくなったりしたら、いったんこのソフトを閉じ、呼び出し直してから使ってください。
　なお、ＳＱＬ命令で削除や書換を行うと元に戻せませんので、UPDATE命令やDELETE命令を打ち込むときには、バックアップを取っておくなり、よく認識した上で操作しましょう。
　また、ACCESSソフトそのものでデータベースを開いておいて、このソフトで同じデータベースを操作しても動作します。
　要領が分かったら、独自のアクセスデータベース（.mdbファイル）を作って実習してみましょう。
